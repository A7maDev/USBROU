USB ROU: USB Read-Only Utility
=====================

Enable or disable reading USB devices on Windows

----------

Requirements
---------
- Windows Vista/7/8
- Windows .NET Framework 4.5 [Download Link](http://www.microsoft.com/en-us/download/details.aspx?id=30653)
